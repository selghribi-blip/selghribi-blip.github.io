# دورة الربح بالذكاء الصناعي المؤتمت للطلاب

<div dir="rtl">

## نظرة عامة

**العنوان الكامل:** دورة الربح بالذكاء الصناعي المؤتمت للطلاب — من الصفر إلى أول دخل رقمي حقيقي

**المنصة:** Udemy

**مدة الدورة التقديرية:** 6–8 ساعات (10 وحدات)

**اللغة:** العربية

**المستوى:** مبتدئ إلى متوسط

---

## فكرة الدورة

تُدرِّب هذه الدورة الطالب العربي الشاب على توظيف أدوات الذكاء الاصطناعي والأتمتة لإنشاء مصادر دخل رقمية حقيقية، دون الحاجة إلى رأس مال أو خبرة برمجية مسبقة. تمشي الدورة خطوة بخطوة: من فهم الأدوات إلى التسليم الأول ثم بناء عمل مستدام.

---

## الجمهور المستهدف

### الملف الشخصي للطالب المثالي

| الخاصية | التفاصيل |
|---------|----------|
| **الفئة العمرية** | 17–28 سنة |
| **المستوى الدراسي** | طالب جامعي أو خريج حديث أو ثانوي في السنة الأخيرة |
| **المستوى التقني** | مستخدم عادي للهاتف والحاسوب — لا يشترط الكود |
| **اللغة** | عربي (يفهم بعض المصطلحات الإنجليزية التقنية البسيطة) |
| **الدخل الحالي** | لا يوجد أو دخل محدود (طالب أو باحث عن عمل) |
| **الوقت المتاح** | 1–3 ساعات يومياً |
| **الاتصال بالإنترنت** | اتصال عادي — الجوال أو لابتوب |
| **الهدف الرئيسي** | كسب أول $100–$500 شهرياً من الإنترنت |

### نقاط الألم الشائعة التي تعالجها الدورة

- "أسمع عن الذكاء الاصطناعي كثيراً لكنني لا أعرف كيف أستفيد منه."
- "كل الدورات بالإنجليزية وصعبة وغالية."
- "لا أملك مال لبدء مشروع."
- "أريد دخلاً إضافياً بجانب دراستي."
- "لا أعرف من أين أبدأ في العمل الحر."

### الطالب الذي لا تناسبه هذه الدورة

- من يبحث عن برمجة احترافية أو تعلم آلي متقدم (Machine Learning)
- من يتوقع ثروة بدون جهد
- من ليس لديه حاسوب أو هاتف ذكي مع اتصال إنترنت
