# Email Extractor — Chrome Extension (MV3)

Extract unique email addresses from the current web page — or just from a text selection — and export them as **CSV** or **JSON**.

## Features

| Feature | Description |
|---|---|
| **Scan page** | Extract all emails from the full page text (+ optional `mailto:` links) |
| **Scan selection** | Extract emails only from the text currently highlighted on the page |
| **Copy all** | Copy every found address to the clipboard |
| **Download CSV** | Export emails as a one-column CSV (`email` header) |
| **Download JSON** | Export emails as a JSON array `{ "emails": [...] }` |
| **Filter** | Live-filter the results list |
| **Dark / Light theme** | Toggle via the ☀/🌙 button in the header, persisted per session |
| **Include mailto links** | Toggle to include / exclude `mailto:` href values in page scans |

## Scan Selection — How it works

1. On any web page, **highlight** the text you want to scan.
2. Click the extension icon to open the popup.
3. Click **"Scan selection"**.
   - If text is selected → emails found in that text are extracted and displayed.
   - If **no text is selected** → a clear info message is shown and existing results are **not** overwritten.

## Installation (Load unpacked)

```bash
cd extensions/email-extractor
npm install
npm run build:css     # generates tailwind.css
```

1. Open Chrome and go to `chrome://extensions`.
2. Enable **Developer mode** (top-right toggle).
3. Click **Load unpacked**.
4. Select the `extensions/email-extractor/` folder.

## Build pipeline

| Command | What it does |
|---|---|
| `npm run build:css` | Builds and minifies `tailwind.css` from `src/input.css` |
| `npm run watch:css` | Rebuilds on every change (development) |

Required files before loading:

```
extensions/email-extractor/
  manifest.json
  background.js
  content.js
  popup.html
  popup.js
  tailwind.css        ← generated by npm run build:css
  icons/
    icon16.png
    icon48.png
    icon128.png
```

## Packaging for the Chrome Web Store

1. Run `npm run build:css` to produce `tailwind.css`.
2. Create a ZIP of the extension folder (**exclude** `node_modules`, `src/`, config files):

```bash
zip -r email-extractor.zip . \
  --exclude "*/node_modules/*" \
  --exclude "*/src/*" \
  --exclude "package*.json" \
  --exclude "tailwind.config.js" \
  --exclude "postcss.config.js"
```

3. Upload to the [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole).
4. In your privacy disclosure note: *"This extension processes page content locally, only when the user clicks Scan. No data is sent to any external server."*

## Permissions

| Permission | Why |
|---|---|
| `activeTab` | Access the currently active tab when the user clicks the extension |
| `scripting` | Inject `content.js` on demand (not always-on) |
| `downloads` | Save CSV / JSON files to disk |

No remote resources are loaded at runtime.

## Privacy

- All email extraction happens **locally** in the browser.
- No data is transmitted to any server.
- The extension does not store or track browsing history.
