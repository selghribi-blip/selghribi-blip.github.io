# Email Extractor — Chrome Extension (Manifest V3)

Extract unique email addresses from the current web page and export them as **CSV** or **JSON**.  
Fully offline — no data is sent to any server.

---

## Features

| Feature | Details |
|---------|---------|
| **Scan page** | Extracts emails from visible text and `mailto:` links |
| **Dark / Light theme** | Toggle in popup header; persists across opens; defaults to system preference |
| **Download CSV** | One email per row, UTF-8 |
| **Download JSON** | Includes `pageUrl`, `pageTitle`, `extractedAt` (ISO 8601), and `emails[]` array |
| **Ignore generic emails** | Toggle to filter out `info@`, `support@`, `sales@`, etc. — editable prefix list, persisted in storage |
| **Filter results** | Live search box to narrow displayed results |
| **Copy all / Copy single** | Clipboard shortcuts |
| **No remote code** | All JS/CSS is bundled locally — Chrome Web Store friendly |

## Getting started (Load Unpacked — Developer mode)

### 1. Build the CSS

```bash
cd extensions/email-extractor
npm install
npm run build:css
```

This runs Tailwind CSS CLI and outputs `tailwind.css` (minified, ~14 KB).

> Re-run `npm run build:css` (or `npm run watch:css` during development) whenever you edit `popup.html` or `popup.js`.

### 2. Load the extension in Chrome

1. Open **Chrome** → `chrome://extensions`
2. Enable **Developer mode** (top-right toggle)
3. Click **Load unpacked**
4. Select the `extensions/email-extractor/` folder

The extension icon appears in your toolbar. Click it on any page that has email addresses.

## Packaging for the Chrome Web Store

1. **Build CSS** (see above) — make sure `tailwind.css` is up to date.
2. **Create a ZIP** (from the `extensions/email-extractor/` folder):
   ```bash
   # macOS / Linux
   cd extensions/email-extractor
   zip -r email-extractor.zip . \
       --exclude "*.DS_Store" \
       --exclude "node_modules/*" \
       --exclude "src/*" \
       --exclude "tailwind.config.js" \
       --exclude "package*.json"
   ```
   On Windows: right-click the folder → *Send to → Compressed (zipped) folder*, then remove `node_modules/` first.

3. Go to the [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
4. Click **New item** → upload `email-extractor.zip`
5. Fill in the listing:
   - Name, short description, detailed description
   - Screenshots (1280 × 800 recommended)
   - Privacy disclosure: *"This extension scans the current page when the user clicks Scan. No data is transmitted to external servers."*
6. Submit for review.

## Permissions used

| Permission | Why |
|------------|-----|
| `activeTab` | Read the URL and title of the current tab |
| `scripting` | Inject content script on demand |
| `downloads` | Trigger CSV / JSON file download |
| `storage` | Persist theme preference and filter settings |
| `host_permissions: <all_urls>` | Allow injection on any page the user opens |
