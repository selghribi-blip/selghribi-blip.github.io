## 🚀 كيفية التشغيل | How to Run

<div style="background:#1a3a5c; color:#fdf6ec; border-radius:12px; padding:24px; margin:24px 0; direction:ltr; text-align:left;">

**1. Install requirements | تثبيت المتطلبات**
```bash
pip install -r csv_cleaner/requirements.txt
```

**2. Run the app | تشغيل التطبيق**
```bash
streamlit run csv_cleaner/app.py
```

**3. Open in browser | فتح المتصفح**  
The app opens automatically at `http://localhost:8501`
