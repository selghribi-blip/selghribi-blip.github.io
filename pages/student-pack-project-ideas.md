# 💡 3 أفكار مشاريع صغيرة مربحة (2–4 أسابيع)

> هذه الأفكار موجّهة لمستخدم **Windows** على مستوى **مبتدئ–متوسط**، مع استخدام أدوات **GitHub Student Developer Pack** المجانية.

---

## 🥇 الفكرة الأولى: قوالب CV / رسائل التقديم (Resume & Cover Letter Templates)

### ماذا تفعل؟
تبني مجموعة من قوالب السيرة الذاتية (CV) ورسائل التقديم (Cover Letters) بتصاميم نظيفة واحترافية جاهزة للتعديل.

> **Simple English:** You make CV and cover letter templates. People buy them and edit their own info. Easy to sell.

### كيف تبيعها؟
- **Gumroad**: ارفع الحزمة كـ PDF / DOCX وضع سعراً بين $5–$15.
- **Fiverr**: اعرض "خدمة تخصيص CV" — تبيع القالب + تعديل بسيط لكل طلب.
- **Creative Market** أو **Etsy** (للقوالب الرقمية).

> **Simple English:** Upload to Gumroad. Set price $5–$15. Done. People pay and download.

### أدوات GitHub Student Developer Pack المقترحة
| الأداة | الاستخدام |
|--------|-----------|
| **GitHub Pages** | نشر صفحة عرض للقوالب مجاناً |
| **Namecheap (نطاق مجاني)** | نطاق احترافي لعرض عملك |
| **GitHub Copilot** | مساعدة في كتابة HTML/CSS للقوالب |
| **Canva for Education** | تصميم القوالب البصرية |

## 🥈 الفكرة الثانية: سكربت أتمتة بسيط لتنظيف بيانات Excel (Data Cleaning Script)

### ماذا تفعل؟
تكتب سكربت Python بسيطاً يأخذ ملف Excel/CSV فيه بيانات فوضوية (أرقام، أسماء، إيميلات) ويُخرج ملفاً نظيفاً مرتباً جاهزاً للاستخدام.

> **Simple English:** You write a Python script. It cleans messy Excel files. Freelancers and small businesses need this a lot.

### كيف تبيعه؟
- **Fiverr**: خدمة "I will clean your Excel/CSV data" بسعر $10–$30 للملف.
- **Gumroad**: بيع السكربت نفسه كـ `.py` + تعليمات الاستخدام بـ $5–$20.
- **Upwork / Freelancer**: مشاريع "data entry cleaning" كثيرة ومتكررة.

> **Simple English:** Sell on Fiverr. Offer to clean one file for $10. Fast money.

### أدوات GitHub Student Developer Pack المقترحة
| الأداة | الاستخدام |
|--------|-----------|
| **GitHub Copilot** | يساعدك تكتب سكربت Python بسرعة |
| **GitHub Pages** | صفحة عرض الخدمة / الأداة |
| **GitHub Actions** | تشغيل الأداة تلقائياً عند رفع ملف |
| **MongoDB Atlas (مجاني)** | تخزين سجل الطلبات إن أردت SaaS صغير |

## 🥉 الفكرة الثالثة: موقع Jekyll جاهز لمحترف حر (Freelancer Portfolio Template)

### ماذا تفعل؟
تبني قالب Jekyll احترافي لبورتفوليو شخصي مُخصَّص للمحترفين الحرّين (مصممين، مطورين، كتّاب). كل شيء جاهز — فقط غيّر المعلومات.

> **Simple English:** You build a portfolio website template with Jekyll. Freelancers buy it to show their work online. Very easy to set up for them.

### كيف تبيعه؟
- **Gumroad**: بيع القالب كـ ZIP بسعر $10–$25.
- **GitHub Marketplace / Themes**: إذا فتحته مفتوح المصدر مع خيار "Pro" مدفوع.
- **Fiverr**: "I will set up your portfolio website" بسعر $20–$50 (تستخدم القالب لكل طلب).

> **Simple English:** Make a nice template once. Sell it many times. Very good business model.

### أدوات GitHub Student Developer Pack المقترحة
| الأداة | الاستخدام |
|--------|-----------|
| **GitHub Pages** | استضافة القالب مجاناً للعرض والتجربة |
| **Namecheap (نطاق مجاني)** | نطاق لصفحة العرض |
| **GitHub Copilot** | مساعدة في كتابة SCSS/HTML/Liquid |
| **Bootstrap Studio (رخصة طالب)** | تصميم سريع للقالب |

## ⭐ الفكرة المختارة لك (Windows + مبتدئ–متوسط + إنجليزية ضعيفة)

> **الفكرة المختارة: الفكرة الأولى — قوالب CV/Cover Letter**

**لماذا هي الأفضل لك؟**
- لا تحتاج سيرفر أو استضافة معقدة — فقط ملفات.
- تعمل 100% على Windows بدون إعداد معقد.
- تبيعها على Gumroad بدون كود خلفي (backend).
- الطلب عليها مرتفع دائماً — كل طالب ومحترف يحتاجها.
- يمكنك البدء خلال يوم واحد فعلياً.

> **Simple English:** This idea is the easiest to start. No server needed. No hard code. Just design + upload + sell.

## 🗺️ خارطة الطريق — أسبوعان (مهمة بمهمة)

> كل المهام بإنجليزية بسيطة وجمل قصيرة.

### 🗓️ الأسبوع الأول — بناء المنتج

**Day 1 – Setup**
- Install VS Code on Windows.
- Create a new GitHub repo called `cv-templates`.
- Enable GitHub Pages for this repo.

**Day 2 – Research**
- Search Google: "best CV templates 2024".
- Write down 3 styles you like: classic, modern, minimal.
- Look at Gumroad: search "CV template" — see what sells.

**Day 3 – Design Template 1 (Classic)**
- Open VS Code.
- Create `template-classic.html`.
- Use GitHub Copilot: type `<!-- simple classic CV layout -->` — it will help you.
- Add: name, photo, skills, experience, education sections.

**Day 4 – Design Template 2 (Modern)**
- Create `template-modern.html`.
- Use a different color scheme (blue or green).
- Keep it clean and simple.

**Day 5 – Design Template 3 (Minimal)**
- Create `template-minimal.html`.
- Black and white. Very simple. One page only.

**Day 6 – Cover Letter**
- Create `cover-letter.html`.
- Add: date, name, body, signature sections.
- Make it match each CV style (copy and adjust colors).

**Day 7 – Test & Fix**
- Open each template in your browser.
- Check: does it look good? Is text readable?
- Fix any broken layout.
- Ask GitHub Copilot to fix CSS issues.

---

### 🗓️ الأسبوع الثاني — البيع والنشر

**Day 8 – Export to PDF**
- Open each HTML in Chrome.
- Press `Ctrl+P` → Save as PDF.
- Name files: `cv-classic.pdf`, `cv-modern.pdf`, `cv-minimal.pdf`.

**Day 9 – Create Product Package**
- Make a folder: `cv-templates-pack/`.
- Put all PDFs + HTML files inside.
- Add a file: `README.txt` — write how to edit the template.

> **Simple English:** "Open the HTML file in VS Code. Change the text. Save. Print as PDF. Done."

**Day 10 – Zip & Upload to Gumroad**
- Zip the folder: `cv-templates-pack.zip`.
- Go to [gumroad.com](https://gumroad.com) — create free account.
- Create a new product. Upload the ZIP.
- Set price: $9.

**Day 11 – Write Product Description**
- Use GitHub Copilot or Google Translate to write the description.
- Title: "3 Professional CV Templates + Cover Letter (HTML + PDF)"
- Add 1–2 screenshots of your templates.

**Day 12 – Create GitHub Pages Preview**
- Push your HTML templates to the GitHub repo.
- Enable Pages → your templates are live at `yourusername.github.io/cv-templates`.
- Add the link to your Gumroad product as "Preview".

**Day 13 – Fiverr Gig**
- Create a Fiverr account.
- Create gig: "I will customize a professional CV template for you".
- Set price: $10 (Basic), $20 (with photo and custom colors).
- Add your GitHub Pages link as portfolio.

**Day 14 – Share & Get First Sales**
- Share on LinkedIn (simple post: "I made 3 CV templates for $9 — link below").
- Share on Reddit: r/learnprogramming, r/cscareerquestions.
- Ask 1–2 friends to buy or review.

> **Simple English:** Tell people about your product. Share the link. Wait for first sale. Improve based on feedback.
