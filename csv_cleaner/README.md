# 🧹 أداة تنظيف CSV/Excel | CSV/Excel Data Cleaner

أداة بسيطة ومجانية لتنظيف ملفات CSV و Excel — واجهة عربية + English  
A simple free tool to clean CSV and Excel files — bilingual Arabic + English UI

---

## ⚡ تشغيل سريع على Windows | Quick start on Windows

### 1. تثبيت المتطلبات | Install requirements

```bat
pip install -r csv_cleaner/requirements.txt
```

### 2. تشغيل التطبيق | Run the app

```bat
streamlit run csv_cleaner/app.py
```

سيفتح المتصفح تلقائياً على `http://localhost:8501`  
Your browser will open automatically at `http://localhost:8501`

---

## ✅ ما تفعله الأداة | What the tool does

| الميزة (AR) | Feature (EN) |
|---|---|
| حذف الصفوف الفارغة | Remove empty rows |
| حذف التكرارات | Remove duplicate rows |
| إزالة المسافات الزائدة | Trim extra spaces |
| إصلاح تنسيقات التواريخ | Fix date column formats |
| حذف الأعمدة الفارغة كلياً | Remove fully empty columns |
| معاينة الملف قبل وبعد | Preview before & after |
| تحميل الملف المنظّف + تقرير | Download cleaned file + report |

## 💡 نصائح | Tips

- **بالعربية:** الواجهة تعرض العربية والإنجليزية معاً دائماً — لا يوجد زر تبديل.
- **In English:** The UI always shows both Arabic and English together — no toggle needed.
- يمكنك تحميل ملف كبير حتى 200 MB | You can upload files up to 200 MB.
