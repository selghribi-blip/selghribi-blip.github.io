## 📦 القوالب المتاحة | Available Templates

### 🌐 قوالب مواقع | Website Templates

| القالب | الوصف | التقنيات |
|--------|-------|---------|
| **Jekyll Blog** | قالب مدونة Jekyll بالعربية + RTL | Jekyll, SCSS |
| **Portfolio Page** | صفحة بورتفوليو بسيطة | HTML, CSS |
| **Landing Page** | صفحة هبوط احترافية | HTML, CSS, JS |

### 🤖 قوالب GitHub Actions | GitHub Actions Templates

| القالب | الوصف |
|--------|-------|
| **deploy-pages.yml** | نشر Jekyll على GitHub Pages |
| **auto-tweet.yml** | نشر تلقائي على Twitter |
| **weekly-report.yml** | تقرير أسبوعي تلقائي |

### 🐍 Python Scripts Templates

| السكريبت | الوصف |
|---------|-------|
| **og-generator.py** | مولد صور Open Graph |
| **newsletter.py** | مولد النشرة البريدية |
| **social-poster.py** | ناشر السوشال ميديا |

## 📥 كيفية الاستخدام | How to Use

1. انتقل إلى ملف القالب الذي تريده
2. انسخ محتواه أو حمله مباشرة
3. عدّل وفق احتياجاتك

## 💡 طلب قالب | Request a Template

هل تريد قالباً محدداً؟ افتح Issue بعنوان `[TEMPLATE] اسم القالب`

Want a specific template? Open an Issue with title `[TEMPLATE] Template Name`
